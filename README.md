# Amber Schematic

A warm dark theme inspired by [Faulty Schematics](https://github.com/Ema0/faulty-schematics-theme).

![Example screenshot](/amber-schematic/images/example.png)

## About

Amber Schematic brings the retro-futurist sci-fi aesthetic of the original Faulty Schematics theme into a comfortable dark mode environment. Featuring warm amber and golden brown tones with strategic vibrant orange accents, this theme evokes the feeling of examining vintage schematics under a warm desk lamp - perfect for late-night coding sessions.

The theme preserves the distinctive character of the original while adapting it for dark mode with a cohesive warm palette that's easy on the eyes.

## Features

- **Warm Amber Palette**: Rich browns and golden tans create a cozy, vintage aesthetic
- **Strategic Orange Accents**: Vibrant `#ff4f00` for cursor and active borders; warmer `#e6915f` for badges and highlights
- **Enhanced Contrast**: Brightened warm tones for improved readability without sacrificing comfort
- **Vintage Character**: Maintains the retro-futurist feel of the original
- **Light Status Bar**: Clear visibility with warm neutral tones

## Color Philosophy

The theme uses a carefully curated warm palette:

- **Backgrounds**: Deep warm browns (`#1a1510`, `#262017`)
- **Text**: Bright warm creams and tans (`#e8dfd4`, `#d8c4a8`)
- **Syntax**: Golden ambers and caramels for visual interest
- **Accents**: Strategic use of vibrant and warm oranges
